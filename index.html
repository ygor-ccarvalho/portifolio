GitHub API

Neste projeto, explorei a API GitHub API para extrair dados de usuário, incluindo o nome, a foto de perfil e os 10 últimos repositórios. Adotei certificações para proporcionar uma experiência mais robusta, como a exibição de mensagens personalizadas em diferentes cenários. Se o usuário não existir, uma mensagem informativa é apresentada. Em caso de campos vazios, é exibido um alerta solicitando o preenchimento adequado. Caso o usuário não possua nome ou descrição, uma mensagem específica é inserida. Além disso com HTML e CSS, busquei criar um layout agradável e responsivo.
